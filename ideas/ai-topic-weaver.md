# AI Topic Weaver

**Short Description:**  
A chat system that automatically splits a user’s message into multiple topics and creates separate conversation threads for each. Example: “Help with Python and low-carb meals” → Coding thread + Diet thread.

**Why it's interesting:**  
People mix many topics in one chat. This creates clean, parallel, self-organizing AI conversations.
